# igame-pc-guide

## 项目简介

igame-pc-guide 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面以静态文件形式存在，不针对任何单个域名或具体网站，旨在提供一种轻量、简洁的页面存储与分发方式。

## 目录说明

```
igame-pc-guide/
├── README.md          # 本文件
├── index.html         # 主入口页面（可选）
└── pages/             # 存放独立 HTML 页面
    ├── page1.html
    ├── page2.html
    └── ...
```

- `index.html`：可选的首页或导航页，用于快速跳转至各个归档页面。
- `pages/`：核心目录，存放所有独立 HTML 页面文件。每个页面均为完整、自包含的静态 HTML。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不依赖外部服务器端环境。
- 页面内容遵循通用 Web 标准，可在主流浏览器中直接打开使用。
- 归档页面不包含任何跟踪、广告或外部资源依赖（除非明确在页面内声明）。
- 仓库不保证页面内容与任何特定域名或网站存在关联。

## 维护说明

- 本仓库由维护者不定期更新，用于收录或整理有价值的独立 HTML 页面。
- 欢迎通过 Issue 提交页面归档建议，但请注意仓库不保证接受所有请求。
- 不建议直接修改他人提交的页面内容，如有修正需求请通过 Issue 讨论。
- 仓库可能调整目录结构或页面命名规范，以保持整洁与可维护性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可将仓库部署至任何静态托管服务（如 GitHub Pages）以供在线访问。
